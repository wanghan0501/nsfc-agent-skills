# NSFC 2026年度申报政策速查

> 具体政策以基金委当年指南为准。

---

## 2026年度主要政策变化

### 申请书结构改革（"瘦身提质"）

正文原则上**不超过30页**。

**面上/青C等项目结构**：
```
（一）立项依据
（二）研究内容
（三）研究基础
（四）其他需要说明的情况
```

**重大研究计划等项目结构**：
```
（一）重点阐述内容（建议不超过5000字）
  1. 研究内容、必要性及与重大研究计划的关系
  2. 关键科学问题、研究方案及可行性分析
  3. 年度研究计划及预期研究成果
（二）研究基础与工作条件
（三）其他需要说明的情况
```

### 代表作制度变化（2026 重点）

- 代表作需**逐篇说明个人贡献情况**（2026 年新要求）
- 优先选择**第一作者 / 共同第一作者 / 通讯作者**成果
- 代表作要与申请选题**强关联**——每篇都应能映射到某项研究内容
- 在附件备注 / 研究基础中说明每篇与本课题的关系

**个人贡献描述写法（可套用）**：
```
本人为该成果第一作者（或共同第一作者，标注 #）/ 通讯作者（标注 *），
负责 [算法设计 / 模型实现 / 数据构建 / 实验验证] 等核心工作，
该成果为本项目研究内容 [X] 提供了 [模型 / 数据 / 方法] 基础。
```

**代表作—选题关联自查**：
- ✅ 每篇代表作都点明"支撑研究内容 1 / 2 / 3 中的哪一项"
- ✅ 作者位置（一作 / 共一# / 通讯*）清晰标注
- ✅ 前期方法/数据如何被本项目直接复用或延续，已说明
- ❌ 避免堆砌与选题无关的"凑数"高分论文

### AI 使用规范

**合规边界**：

| 允许（辅助性） | 禁止（生成性 / 未核实） |
|----------------|--------------------------|
| ✅ 辅助跟踪研究动态、检索整理参考文献 | ❌ 用 AI 直接生成申请书正文 |
| ✅ 辅助语言润色、翻译、格式排版 | ❌ 用 AI 生成研究方案 / 创新点等核心内容 |
| ✅ 辅助文献摘要归纳（人工复核后采用） | ❌ 使用未经核实的 AI 生成内容 |
| ✅ AI 生成**示意性/背景性插图**（如领域概念图，须声明） | ❌ 用 AI 图承载机理/结构/真实数据等科学结论 |

**铁律**：
- ✅ 必须**人工核实** AI 输出的真实性和准确性（文献、数据、结论）
- ✅ 必须在申请书中**全面如实声明** AI 使用情况并标识
- ⚠️ AI 整理文献时会"一本正经地胡说八道"（伪造 DOI、虚构作者）——务必逐条核实
- ⚠️ **投稿中/预印本、私有数据等 AI 检索不到的内容，只能人工撰写与核实**

**AI 使用情况声明范例（脱敏，可参考改写）**：
```
本申请书所有研究背景、科学问题、研究内容、研究方案、创新点及研究基础
均由申请人独立撰写完成。在准备过程中，申请人仅将生成式人工智能工具用于
辅助性工作，具体包括：（1）辅助检索与初步整理公开发表的参考文献；
（2）对个别段落进行语言润色与表述优化。上述 AI 辅助产生的所有信息
（含文献条目、事实陈述）均经申请人逐项人工核实，确保真实、准确。
申请书的核心学术观点、技术路线与结论均为申请人原创，未使用 AI 直接生成。
```
> 声明应放在申请书要求的相应位置，并按当年指南要求的格式标识。

## 项目类型与定位

### 青年科学基金C类（青C）
| 项目 | 说明 |
|------|------|
| **定位** | 人才类项目，重点评价**创新潜力** |
| **年龄** | 男性35岁以下，女性40岁以下 |
| **期限** | 3年 |
| **经费** | 约30万元 |
| **限制** | 同层次人才计划同一年度只能申请一项 |
| **新变化** | 2026年交叉科学部新增受理青C；部分群体（如博士研究生）也可申请（具体资格以当年指南为准） |

**青C 申请人常见情形补充**：
- 博士后 / 在站人员可作为申请人（须依托单位同意、无身份冲突）
- 依托**全国重点实验室 / 国家重点实验室**的，应在"研究基础—工作条件"中
  专章说明研究基地的计算与实验平台落实情况（青C 评审看重可行性支撑）
- 青C 期限 **3 年**，预期成果（论文/人才培养）规模应与 3 年期限匹配，
  不宜照搬面上(4年)的成果体量

### 面上项目
| 项目 | 说明 |
|------|------|
| **定位** | 全面均衡布局，瞄准科学前沿，促进学科发展，激励原始创新 |
| **期限** | 4年 |
| **经费** | 约50-60万元 |
| **合作** | 允许合作单位不超过2个 |
| **评审** | 分"自由探索类"和"目标导向类"两套评审要点 |

### 重点项目
| 项目 | 说明 |
|------|------|
| **限项** | 近年趋于收紧（牵头+承担合计或限一项）；以当年指南为准 |
| **经费** | 约260万元+ |
| **期限** | 4-5年 |

### 重大研究计划
| 项目 | 说明 |
|------|------|
| **特点** | 有指南，需要"打榜" |
| **重点支持** | 4年，约260-400万元 |
| **要求** | 需阐述与计划总体目标的关系 |

### 联合基金
| 项目 | 说明 |
|------|------|
| **特点** | 有指南，需对接甲方需求 |
| **类型** | 区域创新发展联合基金等 |
| **重点** | 4年，约260万元 |
| **关键** | 理解甲方意图，体现协同创新 |

### 其他新变化（趋势性信息，以当年指南公布为准）
- 交叉学部新增工具领域支持（T05代码）
- 拟设/试点**青A赛道**（35岁以内）
- 拟设/试点**科学工具赛道**
- 试点重大非共识项目遴选

> ⚠️ 以上为近年改革趋势与试点方向，具体赛道是否开放、口径如何，**务必以基金委当年正式发布的指南为准**。

## 限项规定要点

- 同层次人才计划同年度只能申请一项
- 重点项目牵头+承担合计限一项
- 同一年度同一国家科技人才计划只能承担一项
- 具体限项规定以当年指南为准
- **建议逐字阅读指南中的申请须知**

## 评审流程

```
形式审查/初审 → 通讯评审（函评，通常5-7位专家）→ 会议评审
```

- 评审主体从小同行向**大同行**转变
- 评审时间极端约束（经验上约20分钟/份）
- 不予受理后可申请复审（成功率经验上极低，逐年不同；以当年规定为准）

---

## 申请代码与学部

- **9个科学部** + 1个国际科研资助部
- 第一申请代码决定送审的处和会评专家
- 关键词匹配函评专家
- **选择学部要考虑**：哪个学部的专家更能欣赏你的工作

---

## 科研诚信底线

**零容忍**：
- ❌ 不得抄袭、剽窃、伪造数据
- ❌ 不得购买/代写申请书
- ❌ 不得打招呼、围会
- ❌ 不得探听评审信息
- ❌ 不得向评审专家提供礼品礼金
- 违规后果严重（通报、取消资格等）

---

## 常见不予受理原因

1. 限项规定违反
2. 申请条件不符（年龄 / 职称 / 身份）
3. 材料不完整
4. 格式严重不规范（擅自改动提纲标题、超页数）
5. 信息虚假（含未核实的 AI 生成内容、伪造文献）
6. 图片打不开 / 乱码
7. "其他需要说明的情况"未如实填写（同年申请冲突、单位不一致等）

**提交前自查清单**：
- ✅ 所有图片、附件可正常打开（自己 demo 一遍）
- ✅ 正文未删改提纲标题及括号文字，未超 30 页
- ✅ "其他需要说明的情况"逐条作答（无冲突也要明确写"无"）
- ✅ 参考文献链接（EB/OL）逐条可访问
- ✅ AI 使用情况已如实声明、内容已人工核实
